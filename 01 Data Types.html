01 Data types¶

1.1 Categorising data points using NOIR¶

You are constantly dealing with data. You might be looking at the number of students enrolled in a district, the attendance rate of a specific school, the average test score in a subject, or the percentage of teachers with advanced degrees. Your data can be categorised as Nominal, Ordinal, Interval, and Ratio (NOIR), as explained in the table below.

                                                                                                                                         
Data TypeDescriptionEducational Examples
NominalCategories with no order. You can count how many data points fall into each category, but you can't perform mathematical operations like addition or subtraction on them. "Male" is not "greater than" or "less than" "Female.". A type of qualitative data.Student gender (Male, Female),
School district name,
Type of school (Public, Private, Charter)
OrdinalCategories with a meaningful order, but unclear or inconsistent intervals. The order of the data matters, but the gaps between values does not. Think of a ranking. You know that one category is "more" or "better" than another, but you don't know by how much. For example, a teacher rating of 4 is better than a 3, but the difference between 4 and 3 might not be the same as the difference between 2 and 1. A type of qualitative data.Teacher satisfaction rating (1-5),
School quality rating (A, B, C, D),
Year group (Year 1, Year 2, Year 3)
IntervalConsistent, meaningful intervals between values, but it lacks a true zero point. This means you can add and subtract values, but you can't perform multiplication or division. A true zero means the complete absence of the thing being measured. For example, an IQ score of 120 is 20 points higher than 100, but a score of 0 does not mean a complete lack of intelligence. A type of quantitative data.Standardized test scores (e.g., SAT, GRE),
Calendar years
RatioOrdered data with consistent intervals and a true zero point. The value 'zero' means absence. This is the most informative type of data. It has all the properties of interval data (order and consistent intervals) but also includes a true zero point. This means you can perform all mathematical operations—addition, subtraction, multiplication, and division—and the ratios between values are meaningful. A classroom with 20 students has twice as many as one with 10 because 0 students means the classroom is empty. A type of quantitative data.Number of students in a class,
Teacher salary,
Student attendance days,
School budget

1.3 A Decision Tree for Statistical Tests¶

Now that you understand NOIR data categories, you can start to work out which statistical test will help you understand it, compare groups or predict outcomes and relationships.

We need to learn more about data - specifically, about dependent and independent variables, and parametric and non-parametric data. YOu’ll do that in Chapter 3. By then, you’ll be able to use the more sophisticated decision tree below, which will help you work out what test you need for the vast majority of data you’re likely to analyse (or ask your team to analyse!).

, we can link it directly to the choice of a statistical test. The data type of your variables is a primary factor in determining which test is appropriate. The following decision tree provides a roadmap for selecting the correct test based on your research question and the type of data you have.

![Decision tree](Decision tree.png)

This chart is a powerful roadmap. For instance, if you want to know if there’s a difference in average test scores (a Ratio/Interval variable) between male and female students (two independent groups), the tree guides you to the Independent Samples t-test. If you want to see if a new teaching method improved student scores before and after the intervention, you would use a Paired Samples t-test. The key is to start with your research question and then identify your variables and their NOIR classification to follow the correct path.

Throughout this book, we will work through each of these decision points, using Python to apply the appropriate statistical test and interpret the results in a way that is meaningful for educational policy.
