Statistical Analysis for Education¶

This interactive textbook will help education professionals to use basic statistical analysis tools.

The content is focussed on education data and systems in lower-income countries, and the particular challenges they face.

We will use AI to answer questions, and a programming language called Python.

It assumes high-school maths, and some experience of using data in your day-to-day job, but no other statistics education or training.

Building:¶

To build the document, in the terminal, execute: sphinx-build -M html docs/source/ docs/build/

(this assumes you’ve opened the root folder as Stats-for-EID… you might need to run: cd pmdawes@PadBook Stats-for-EID-textbook % cd ‘/Users/pmdawes/Documents/03 AI and data science/11 Projects/Stats-for-EID-textbook’ )

Pushing to GitHub:

git commit -m “Version Changes” && git push origin main
